#  WEATHER APP 

A simple, real-time weather application built with **HTML, CSS, and JavaScript**.  
Check the weather of any city worldwide and get temperature, humidity, wind speed, and a dynamic weather icon.

![interface](/interface.png)

## Features

- Search the **current weather** for any city  
- Displays **temperature, humidity, and wind speed**  
- Dynamic weather icons for:
  - Clear  
  - Clouds  
  - Rain  
  - Drizzle  
  - Mist  
- Handles **invalid city names** gracefully  
- Clean, responsive design for desktop and mobile

## Usage

- Type the name of a city in the input box.
- Click the search button or press Enter.
- View live weather data, including temperature, humidity, wind speed, and a weather icon.
- If the city name is invalid, an error message will be displayed.

## Technologies

- HTML5 – structure and layout
- CSS3 – styling and responsive design
- JavaScript (ES6+) – API requests, DOM updates, and interactivity
- OpenWeatherMap API – real-time weather data
